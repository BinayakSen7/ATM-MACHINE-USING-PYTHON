# Atm Machine using python

The system ensures that only positive numbers can be deposited or withdrawn, and it prevents withdrawing more than the available balance.

## Features

* View current balance
* Deposit funds (positive numbers only)
* Withdraw funds (positive numbers only, cannot exceed balance)
* User-friendly menu interface
* Looping menu until user chooses to exit

## How to Run

1. open 'Atm\_machine.ipynd'
2. Run the program
3. see the output
